Frequently Asked Questions

Here are some of our FAQs. if you have any other questions you'd like answered please feel free to email us.

What is Bookmark?

In the context of your project/UI design, "Bookmark" means saving a website link so you can come back to it later easily.

How can I request a new browser?

You can install a new browser by downloading it from its official website (e.g., Chrome, Firefox, Opera) and running the installer.

Is there a mobile app?

Yes there are many official mobile apps for both iOS (App Store) and Android (Google Play Store).

What about other Chromium browsers?

There are many other Chromium-based browsers like Brave, Opera, Microsoft Edge, Vivaldi, and Arc, all built on the same engine as Chrome but with different features.
